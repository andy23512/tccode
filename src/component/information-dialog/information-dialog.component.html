Information

This website, TanChord Code (TC Code in short), is an unofficial, experimental editor that allows you to edit the chord library of your CharaChorder device, similar to editing code in a code editor.

TCCL Language

This editor allows you to describe the chord library with a format similar to the chord notation described in the CharaChorder official document. I call this language TanChord Code Library language, TCCL in short. There is a language reference in my blog if you want to learn the language syntax.

Main Procedure

The main procedure in this editor is composed of three parts, loading, editing and saving chords.

{{ 'forms_add_on' | iconGuard }} Loading Chords

You can load your existing chord library from a device, a JSON backup file from the official device manager, a TCCL or TXT file in TCCL format, or a built-in sample. The loaded chord library would be added to the end of the editor content.

{{ 'edit' | iconGuard }} Editing Chords

In the main area of this editor, you can edit the chord library. There are some built-in checkers for duplicated chords and syntax errors, similar to error checkers in some code editors.

{{ 'save' | iconGuard }} Saving Chords

After you finish your edits, you can save the chord library to a device or a TCCL file.

Side Features

{{ 'lock' | iconGuard }} Editor Lock

Make the editor read-only. It's for when you only want to navigate the chord library without making changes.

{{ 'settings' | iconGuard }} Settings

Tweak some settings of this tool, such as the welcome dialog, auto connect, editor key bindings (classic / emacs / vim), and Bopomofo chord conflict detection.

{{ 'article' | iconGuard }} Serial Log

The logging for the Web serial input and output. It's mainly for debugging serial commands.

{{ 'account_tree' | iconGuard }} Abstract Syntax Tree

The abstract syntax tree of the content in the editor. It's mainly for debugging the TCCL grammar, lexer, and parser.
